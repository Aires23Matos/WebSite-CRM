# WebSite-CRM
=======
# WebSite Sistema de Gestão de Clientes 

Este é uma plataforma criado para gerenciar clientes que fazem o pedido de um serviço que a empresa presta.


## Forma de executar o WebSite

Instalar todos requisitos do nodes
````
npm install
````

Executar como desenvolvedor 
````
npm run dev
````
